# Check and prepare nodbi connection object for ctrdata

Check and prepare nodbi connection object for ctrdata

## Usage

``` r
ctrDb(con)
```

## Arguments

- con:

  A database connection object, created with `nodbi`. See section \`1 -
  Database connection\` in
  [ctrdata](https://rfhb.github.io/ctrdata/reference/ctrdata.md).

## Value

Connection object as list, with collection element under root
